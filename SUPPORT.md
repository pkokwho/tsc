### Ask a Question

- **GitHub Discussions** — For general questions and ideas, use [GitHub Discussions](https://github.com/pkokwho/tsc/discussions)
- **GitHub Issues** — For bug reports and feature requests, use [GitHub Issues](https://github.com/pkokwho/tsc/issues)

### Report a Bug

Please use the [Bug Report template](https://github.com/pkokwho/tsc/issues/new?template=bug_report.md) when filing a bug. Include:

- Browser name and version
- Operating system
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

### Request a Feature

Please use the [Feature Request template](https://github.com/pkokwho/tsc/issues/new?template=feature_request.md) when suggesting a feature. Describe:

- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered
