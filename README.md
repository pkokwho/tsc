## Why TSC?

TSC AI Prompt Studio 帮助用户将简单想法快速转换为适用于不同 AI 模型的高质量 Prompt。

支持：

- ChatGPT / 通用模型
- Midjourney
- Flux
- Video AI

## 功能特性

- 多模型 Prompt 生成
- 中英双语支持
- 自动检测浏览器语言
- 深色 / 浅色主题
- 12 种艺术风格
- 可调比例与质量参数
- 一键复制 Prompt
- 响应式设计
- 零依赖部署
- SEO 与社交媒体优化

## 快速开始

```bash
git clone https://github.com/pkokwho/tsc.git
```

直接打开 `index.html` 即可运行。

## GitHub Pages 部署

1. 推送代码到 GitHub
2. Settings → Pages
3. 选择默认分支和根目录
4. 保存并等待部署完成
